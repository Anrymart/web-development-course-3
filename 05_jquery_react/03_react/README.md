## Setup

1. Install Node.js - https://nodejs.org/en/download/
2. Install npm: `npm install -g npm`
3. Check versions:
   ```
   node -v
   npm -v
   ```
4. Create React app:
   ```
   npx create-react-app react-tutorial
   cd react-tutorial
   npm start
   ```
